# Logo5 by SoulFlame

Завършен frontend product за логопедична помощ за деца в Сливен.

## Продуктова идея

Logo5 е система за логопедичен кабинет, която комбинира:

- публична landing страница
- записване на дете
- услуги и цени
- родителски профил
- личен чат за конкретното дете
- библиотека с материали, снимки и видеа
- профил „За Веска“
- админ панел за записвания, съобщения и материали

## Бранд

**Logo5 by SoulFlame**

Logo5 означава 5 стъпки към ясна реч:

1. Оценка
2. План
3. Упражнения
4. Подкрепа
5. Напредък

## Технология

Проектът е статичен single-file frontend в `index.html`, готов за Vercel.

## Deploy във Vercel

1. Vercel → Add New Project
2. Import GitHub repo: `SoulFlameAdmin/logo5`
3. Project name: `logo5`
4. Framework preset: Other
5. Deploy

След deployment адресът трябва да бъде:

`https://logo5.vercel.app`
