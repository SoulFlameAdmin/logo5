# Logo5

Logo5 by Veska Lambova — MVP сайт за логопедична помощ за деца в Сливен.

## Страници
- Записване на дете
- Пишете ми — личен чат за конкретното дете
- Библиотека — снимки, видеа и материали
- За мен — професионален профил на Веска Ламбова

## Deploy
Статичен сайт, готов за Vercel. Главният файл е `index.html`.
